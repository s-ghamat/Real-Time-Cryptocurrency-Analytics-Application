# 📊 **Creative Analytics – Visualisations clés du marché crypto**

> Une suite *intelligente* de charts pour **lire l’état du marché en quelques secondes**, au-delà des chiffres bruts.



## 1️⃣ **Multi-Asset Rhythm**

**Comparaison normalisée** du mouvement de plusieurs cryptos sur une “journée” synthétique.
➡️ Chaque courbe est ramenée sur une échelle **0 → 1** pour comparer le **rythme relatif**.

### 🎯 Objectif

* Lire le **pouls du marché**
* Repérer :

  * Actifs **stables / smooth**
  * Actifs **volatils / nerveux**
  * **Synchronisation** entre actifs (macro / sectorielle)

### 🧠 Intérêt

Permet de *sentir* l’ambiance du marché :

> calme, agité ou coordonné → lecture narrative & instinctive.



## 2️⃣ **Market Temperature**

Thermomètre basé sur la **performance moyenne 24h** du marché.

📈 Gradient **rouge → jaune → vert** + **curseur + valeur numérique**

### 🎯 Objectif

➡️ Répond instantanément à :

> **“Le marché est bullish, neutre ou bearish aujourd’hui ?”**

### 🧠 Intérêt

* Contexte macro ultra rapide
* À consulter **avant d’analyser un actif précis**



## 3️⃣ **Market Cap Galaxy**

Donut + liste des poids dans la capitalisation totale.

### 🎯 Objectif

* Visualiser **qui domine vraiment**
* Savoir si **le marché est concentré ou équilibré**

### 🧠 Intérêt

> Raconte l’histoire structurelle du marché
> → *où se trouve la vraie valeur ?*



## 4️⃣ **Liquidity Pulse**

Bar chart du ratio **Volume / Market Cap** → proxy de la liquidité.

### 🎯 Objectif

* Identifier les actifs **vraiment “vivants”**
* Détecter les **risques cachés** : spreads, slippage…

### 🧠 Intérêt

> Un actif illiquide = risque invisible.
> Ce graphique le rend **immédiatement visible**.



## 5️⃣ **Dominance Buckets**

| Bucket     | Description          |
| ---------- | -------------------- |
| Large Caps | Top 3 par market cap |
| Mid Caps   | Top 4–10             |
| Others     | Longue traîne        |

### 🎯 Objectif

Comprendre la **structure du marché**
→ utile pour **allocation & profil de risque**

### 🧠 Intérêt

> Permet de **penser comme un gestionnaire de portefeuille**.



## 6️⃣ **Correlation Matrix**

Heatmap NxN — corrélation entre les actifs.

| Couleur  | Signification        |
| -------- | -------------------- |
| 🟢 Vert  | corrélation positive |
| 🔴 Rouge | corrélation négative |
| ⚪ Gris   | neutre / faible      |

### 🎯 Objectif

Repérer :

* Actifs synchronisés
* Hedges potentiels
* Indépendances structurelles

### 🧠 Intérêt

> Clé pour la **diversification intelligente**.



## 7️⃣ **Risk–Return Scatter**

| Élément      | Représente             |
| ------------ | ---------------------- |
| X-axis       | Volatilité (synthetic) |
| Y-axis       | Performance 24h        |
| Taille bulle | Market cap             |
| Couleur      | Perf + / –             |

### 🎯 Objectif

Comparer **risque vs rendement**
→ détecter les **outliers intéressants**

### 🧠 Intérêt

> Adaptation crypto d’un classique de la finance 📈



## 8️⃣ **Net Flow Bars**

`Flow ≈ price_change × liquidity`

### 🎯 Objectif

* Estimer la **pression d’achat / vente**
* Différencier :

  * hausse fragile (peu de volume)
  * hausse solide (volume élevé)

### 🧠 Intérêt

> Proxy simple de **“where the money flows”**



## 9️⃣ **Momentum Heatmap**

| 1H    | 4H    | 24H   | 7D    | 30D    |
| ----- | ----- | ----- | ----- | ------ |
| +0.7% | +2.7% | +3.9% | +8.8% | +15.0% |

### 🎯 Objectif

Lire la **cohérence d’une tendance**

* Vert partout → momentum solide
* Divergence → potentiel retournement

### 🧠 Intérêt

> Equivalent crypto du **multi-timeframe prop trading**



## 🔟 **Emerging Movers**

Radar des actifs **hors top 5** qui s’agitent fortement.

Infos affichées :

* Nom + symbole
* Liquidity tag
* % change
* Volume / Market Cap
* Prix actuel

### 🎯 Objectif

Repérer les **nouvelles narratives**
→ zone où naît le *vrai alpha*



## 🚀 **Conclusion**

Ces visualisations offrent :
✔ Lecture rapide du marché
✔ Indicateurs actionnables
✔ Pont entre **tradfi & dynamique crypto**
